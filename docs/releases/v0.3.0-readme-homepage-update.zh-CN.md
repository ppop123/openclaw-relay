# README 首页更新文案（v0.3.0）

`OpenClaw Relay v0.3.0` 已发布。

这是一个以 Web Client 产品化、浏览器身份持久化、真实浏览器 E2E 与文档收口为重点的功能版本。

## 首页摘要建议

- 正式支持范围保持为：`relay/`、`sdk/python/`、`client/`、`plugin/`、`protocol/`
- `v0.3.0` 新增重点：持久化浏览器身份、受口令保护的身份导入导出、saved profiles、preferred agent、chat export、deterministic/live browser E2E
- README 现已改成双语首页，中文在前，英文在后
