# README 首页更新文案（v0.2.1）

`OpenClaw Relay v0.2.1` 已发布。

这是一个以插件生命周期稳定性和真实宿主发版验证为重点的补丁版本。

## 首页摘要建议

- 正式支持范围保持为：`relay/`、`sdk/python/`、`client/`、`plugin/`、`protocol/`
- `v0.2.1` 新增的重点是 plugin lifecycle smoke，覆盖 pair → revoke → re-pair → rotate-token → disable
