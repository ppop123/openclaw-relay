# README 首页更新文案（v0.5.0）

`OpenClaw Relay v0.5.0` 已发布。

这是一个小型补丁版：首页更新重点是把“怎么使用”写得更可复现，并把 agent-only peer capability 的产品边界说得更清楚。

## 首页摘要建议

- 正式支持范围保持为：`relay/`、`sdk/python/`、`client/`、`plugin/`、`protocol/`
- `v0.5.0` 重点修复 peer capability 的耐用性问题，并把 `relay.peer.*` 的 agent-only 边界落实到代码里
- README 现在明确给出可复现的路径：`enable` → `pair --wait` → 浏览器在等待窗口里连接
