# README 首页更新文案（v0.3.1）

`OpenClaw Relay v0.3.1` 已发布。

这是一个小型 hotfix 版本：首页更新重点是把“怎么使用”写得更直接，让第一次进入仓库的用户知道该怎么从 relay、plugin、web client 走通完整链路。

## 首页摘要建议

- 正式支持范围保持为：`relay/`、`sdk/python/`、`client/`、`plugin/`、`protocol/`
- `v0.3.1` 修复了 plugin 真实 relay 集成测试在 CI 上的稳定性问题
- README 现在明确给出“部署 relay → 安装 plugin → 浏览器连接”的使用路径
