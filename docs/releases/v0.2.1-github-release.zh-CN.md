# OpenClaw Relay v0.2.1

`OpenClaw Relay v0.2.1` 现已发布。

这是一个以**网关插件稳定性**、**真实宿主生命周期验证**和**发版收口**为重点的补丁版本，承接 `v0.2.0`。

## 重点更新

- 修复 `openclaw relay pair` 的完成判定，让 CLI 能更稳定地报告配对成功
- 新增真实宿主 lifecycle smoke，覆盖：
  - 初次配对
  - 请求验证
  - revoke
  - 重新配对
  - rotate-token
  - disable
- 改进本地 smoke 隔离策略：每次运行使用全新的 OpenClaw 隔离状态目录
- 为配对完成路径补上宿主桥接回归测试

## 正式支持范围

`v0.2.1` 的正式支持范围保持不变：

- `relay/` — Go relay 服务
- `sdk/python/` — Python 客户端 SDK（Layers 0–2）
- `client/` — Web 参考客户端
- `plugin/` — OpenClaw 网关插件
- `protocol/` — 协议文档、错误码、canonical fixtures

## 不在正式支持范围内

以下内容仍然不属于正式支持面：

- `plugin/` 中超出当前文档定义的 notify 扩展能力
- `sdk/js/` — 尚未实现

## v0.2.1 具体变化

- 收紧插件 CLI 的 pairing 状态判定逻辑
- 将本地发布验证从单一路径 smoke 扩展为 lifecycle smoke
- 在 plugin 改动后继续保持 release contract、protocol examples、docs consistency 检查全绿

## 验证结果

`v0.2.1` 已通过：

- `cd relay && go test -v -count=1 ./...`
- `cd sdk/python && pytest -q`
- `cd client && npm test -- --run`
- `cd plugin && npm ci && npm test && npm run typecheck`
- `python3 scripts/validate-protocol-examples.py`
- `bash scripts/check-doc-consistency.sh`
- `bash scripts/smoke-openclaw-plugin.sh`（本地/人工，真实 OpenClaw runtime）

## 说明

- `v1` 仍然**不提供** forward secrecy
- 插件 lifecycle smoke 仍然保持本地/人工执行，因为托管 CI 不提供真实 OpenClaw runtime

欢迎继续试用、提 issue、做审计和提交 PR。
