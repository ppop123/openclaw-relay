# README homepage update for v0.3.0

`OpenClaw Relay v0.3.0` is now available.

This feature release focuses on web-client productization, persistent browser identity, real-browser E2E coverage, and documentation consistency hardening.

## Suggested homepage summary

- Official support remains: `relay/`, `sdk/python/`, `client/`, `plugin/`, `protocol/`
- New in `v0.3.0`: persistent browser identity, passphrase-protected identity import/export, saved profiles, preferred agent memory, chat export, and deterministic/live browser E2E
- The README homepage is now bilingual, with Chinese first and English second
