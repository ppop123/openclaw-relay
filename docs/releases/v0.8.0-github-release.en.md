# OpenClaw Relay v0.8.0

This release adds a JS SDK, makes the desktop shell official, and tightens admin panel security.

## JavaScript SDK

You can now connect to Relay from Node.js without implementing the protocol yourself. The new `sdk/js/` package covers Layers 0–2:

- Same crypto handshake as the browser client (X25519 + AES-256-GCM)
- Streaming support: `chat.send`, `agents.list`, `system.status`
- Comes with Vitest tests

## Desktop Shell

The desktop shell is now an officially supported component and part of the release gates.

- `npm run sync-client` syncs the web frontend into `desktop/webview-dist/`
- `tauri build --bundles app` produces the local app bundle

## Admin Panel Security

The admin panel now requires a short-lived admin key on top of the existing client approval.

- Generate one via the `/relay_admin` chat command or `openclaw relay admin-key` CLI
- The key lives only in browser memory — close the tab and it's gone

---

For the full support scope and security model, see the [v0.3.0 release notes](https://github.com/ppop123/openclaw-relay/releases/tag/v0.3.0).
