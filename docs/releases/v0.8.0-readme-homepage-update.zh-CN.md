# README 首页更新（v0.8.0）

`OpenClaw Relay v0.8.0` 发布了。

这版加了 JS SDK、桌面壳转正、管理面板多了 admin key 验证。

## 首页摘要

- 正式支持：`relay/`、`sdk/python/`、`sdk/js/`、`client/`、`desktop/`、`plugin/`、`protocol/`
- v0.8.0：Node.js 可以直接用 JS SDK 连 Relay
- v0.8.0：管理面板需要短时 admin key
