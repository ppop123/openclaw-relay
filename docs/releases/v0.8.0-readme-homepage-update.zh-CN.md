# README 首页更新文案（v0.8.0）

`OpenClaw Relay v0.8.0` 已发布。

本次版本新增 JavaScript SDK，桌面壳进入正式发布门禁，并为管理界面引入短时 admin key 二次授权。

## 建议首页摘要

- 正式支持组件包括：`relay/`、`sdk/python/`、`sdk/js/`、`client/`、`desktop/`、`plugin/`、`protocol/`
- `v0.8.0` 交付 Node 端 JavaScript SDK（Layer 0–2）
- `v0.8.0` 为管理面板加入 admin-key 二次授权
