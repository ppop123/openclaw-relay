# README homepage update for v0.5.1

`OpenClaw Relay v0.5.1` is now available.

This patch release brings the web management console to the browser client and hardens relay/plugin safety around admin and peer traffic.

## Suggested homepage summary

- Official support remains: `relay/`, `sdk/python/`, `client/`, `plugin/`, `protocol/`
- `v0.5.1` adds multi-tab chat with an agent selector plus admin panels for skills/config/logs/maintenance
- `v0.5.1` tightens admin access to approved clients and adds periodic invite pruning in the relay
