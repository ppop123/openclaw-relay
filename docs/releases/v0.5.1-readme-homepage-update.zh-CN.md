# README 首页更新文案（v0.5.1）

`OpenClaw Relay v0.5.1` 已发布。

本次补丁版将 Web 管理控制台带到浏览器客户端，并加强了 relay / plugin 在管理与 peer 流量上的防护。

## 建议首页摘要

- 官方支持仍覆盖：`relay/`、`sdk/python/`、`client/`、`plugin/`、`protocol/`
- `v0.5.1` 新增多标签对话与 agent 选择器，并提供技能/配置/日志/维护管理面板
- `v0.5.1` 收紧管理接口访问并在 relay 侧加入 invite 定期清理
