# OpenClaw Relay v0.5.1

v0.5.1 is a patch release that adds the management UI to the web client and hardens relay/plugin safety around admin and peer traffic.

## Web Client

- Multi-tab chat with per-tab agent selection via the new agent selector
- Dashboard overview for system status, agents, and cron jobs
- Admin console panels for skills, config read/write/apply, log tailing, and maintenance actions

## Relay + Plugin Hardening

- Relay now prunes expired peer invites periodically to prevent unbounded growth
- Gateway plugin enforces admin method access to approved clients only
- Strict JSON parsing for gateway CLI output (no substring recovery)
- Decrypted Layer2 payloads now have a size cap before JSON parse
- Peer session frames are processed sequentially to avoid out-of-order stream chunks

## UX Fixes

- The “save this connection” banner now dismisses correctly after save/ignore
