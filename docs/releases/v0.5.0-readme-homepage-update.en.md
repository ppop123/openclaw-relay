# README homepage update for v0.5.0

`OpenClaw Relay v0.5.0` is now available.

This small patch release updates the homepage to make the usage flow reproducible and the agent-only peer boundary easier to understand.

## Suggested homepage summary

- Official support remains: `relay/`, `sdk/python/`, `client/`, `plugin/`, `protocol/`
- `v0.5.0` hardens peer-capability durability and enforces the `relay.peer.*` agent-only boundary in code
- README now shows the reproducible path: `enable` → `pair --wait` → browser connects while the pairing window is open
