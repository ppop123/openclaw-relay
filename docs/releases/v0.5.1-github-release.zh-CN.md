# OpenClaw Relay v0.5.1

v0.5.1 是一个补丁版本，新增了 Web 管理界面，并加强了 relay / plugin 在管理与 peer 流量上的安全防护。

## Web 客户端

- 多标签对话，支持按标签选择 agent（新增 agent 选择器）
- 概览面板：系统状态、agents、定时任务一页查看
- 管理面板：技能、配置读写/应用、日志拉取、维护操作

## Relay 与 Plugin 加固

- relay 定时清理过期 invite，避免长期运行内存增长
- gateway plugin 的管理接口仅允许已批准客户端访问
- gateway CLI 输出改为严格 JSON 解析（不再用子串兜底）
- 解密后的 Layer2 明文新增大小上限，避免超大 JSON 解析
- peer 会话帧按顺序串行处理，避免流式 chunk 乱序

## 体验修复

- “保存此连接”提示在保存/忽略后可正确消失
