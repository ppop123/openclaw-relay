# README homepage update for v0.8.0

`OpenClaw Relay v0.8.0` is now available.

This release introduces the JavaScript SDK, promotes the desktop shell to an official release gate, and adds short-lived admin keys for the management UI.

## Suggested homepage summary

- Official support now includes: `relay/`, `sdk/python/`, `sdk/js/`, `client/`, `desktop/`, `plugin/`, `protocol/`
- `v0.8.0` ships the Node-focused JavaScript SDK with Layer 0–2 protocol support
- `v0.8.0` adds admin-key secondary auth for management panels
