# README homepage update for v0.8.0

`OpenClaw Relay v0.8.0` is out.

This release adds a JS SDK, makes the desktop shell official, and requires an admin key for the management panel.

## Homepage summary

- Officially supported: `relay/`, `sdk/python/`, `sdk/js/`, `client/`, `desktop/`, `plugin/`, `protocol/`
- v0.8.0: Connect to Relay from Node.js with the new JS SDK
- v0.8.0: Admin panel now requires a short-lived admin key
