# README homepage update for v0.3.1

`OpenClaw Relay v0.3.1` is now available.

This is a small hotfix release. The homepage update focuses on making the “how to use it” path explicit so first-time users can see the relay → plugin → browser flow immediately.

## Suggested homepage summary

- Official support remains: `relay/`, `sdk/python/`, `client/`, `plugin/`, `protocol/`
- `v0.3.1` fixes a CI stability issue in the plugin real-relay integration test path
- The README now directly explains the deploy-relay → install-plugin → connect-from-browser flow
