# OpenClaw Relay v0.8.0

这个版本加了 JS SDK，桌面壳转正，管理面板安全性也提升了。

## JavaScript SDK

现在可以在 Node.js 里直接连 Relay 了，不用自己实现协议。新的 `sdk/js/` 包覆盖了 Layer 0–2：

- 加密握手和浏览器端一致（X25519 + AES-256-GCM）
- 支持流式调用：`chat.send`、`agents.list`、`system.status`
- 自带 Vitest 测试

## 桌面壳

桌面壳现在是正式支持组件，发版时会一起跑门禁。

- `npm run sync-client` 把 web 前端同步到 `desktop/webview-dist/`
- `tauri build --bundles app` 打包本地应用

## 管理面板安全

管理面板现在除了客户端审批，还需要一个短时 admin key 才能操作。

- 用 `/relay_admin` 聊天指令或 `openclaw relay admin-key` CLI 生成
- key 只存在浏览器内存里，关掉就没了

完整支持范围与安全模型见 [v0.3.0 release notes](https://github.com/ppop123/openclaw-relay/releases/tag/v0.3.0)。
