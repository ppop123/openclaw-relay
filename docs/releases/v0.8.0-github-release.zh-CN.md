# OpenClaw Relay v0.8.0

v0.8.0 交付了期待已久的 JavaScript SDK，桌面壳进入正式发布门禁，并为管理界面新增短时 admin key 二次授权。

## JavaScript SDK（Node）

- 新增 `sdk/js/`，提供 Node 端 Relay 客户端（Layer 0–2）
- X25519 + HKDF + AES-256-GCM 握手与传输与浏览器参考客户端保持一致
- 支持流式 `chat.send`、`agents.list`、`system.status`
- Vitest 覆盖加密与传输行为

## 桌面壳

- 桌面壳升级为正式支持组件，纳入发布门禁
- 通过 `npm run sync-client` 同步共享前端到 `desktop/webview-dist/`
- 发布构建使用 `tauri build --bundles app` 生成本地应用

## 管理界面安全加固

- 管理面板在已批准客户端基础上新增短时 admin key 校验
- 新增 `/relay_admin` 聊天指令与 `openclaw relay admin-key` CLI
- admin key 仅保存在浏览器内存中，可随时清除

---

完整支持范围与安全模型请参见 [v0.3.0 release notes](https://github.com/ppop123/openclaw-relay/releases/tag/v0.3.0)。
